## Bug 修复 (17:52 - 点击无反应)
- 根因：overlay div 的 `pointer-events: all` + `z-index: 10` 覆盖全屏，但状态栏/调试层放在 overlay 内部，导致消息被吞。
- 修复：状态栏 `#statusBar z-index: 200`、`#debugLog z-index: 300` 放在 overlay 同级 DOM 后面。
- 加了全屏诊断浮层 `#diagOverlay`（出错时自动弹出）、全局 JS 错误捕获、`Babylon.js` 预加载检查、多层 debug 日志。
- 暴露 `window._mgr`、`window._setStatus` 供控制台调试。
